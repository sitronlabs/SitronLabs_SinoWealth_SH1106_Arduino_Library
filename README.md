# Sino Wealth SH1106 Arduino Library
Arduino library for OLED displays based on the Sino Wealth SH1106 controller.

### Supported interfaces
| Interface | Status |
|-----------|:------:|
| I2C, Unbuffered | ✔️ |
| I2C, Buffered | ✔️ |
| SPI, 3-Wires, Buffered | ❌ |
| SPI, 4-Wires, Buffered | ❌ |
